Real Estate Database Performance Comparison
A comprehensive performance comparison between PostgreSQL and MongoDB for real estate data management, featuring data import utilities and multi-scenario performance testing.

Quick Start

Prerequisites

Python 3.7 or higher
PostgreSQL 12+
MongoDB 4.4+

Installation

Clone the repository(Housing.csv)
Install Python dependencies
Database Setup (PostgreSQL and MongoDB)

Usage

Import data to both databases ---> (python) data_importer.py
Run performance tests ---> (python) performance_tester.py
View results ---> performance_results.json
Also console output with summarized results
